Fun and Games

Things you may enjoy:

Turtle Blocks

Turtle Blocks Javascript is an activity with a Logo-inspired graphical "turtle" that draws colorful art based on snap-together visual programming elements. Its "low floor" provides an easy entry point for beginners. It also has "high ceiling", for programming, graphics, mathematics, and Computer Science features that will challenge the more adventurous student.

A guide to Turtle Blocks JS is available. (pdf)

TortugaBloques

TortugaBloques JavaScript es una actividad con una "tortuga" gráfica del Logo tipo de inspiración que dibuja el arte colorido basado en elementos de programación visual de traba a presión. Su "piso bajo" proporciona un punto de entrada fácil para los principiantes. También tiene "techo alto" para de programación, gráficos, matemáticas y ciencias de la computación cuenta que pondrá a prueba el estudiante más aventurero.

Una Guía para el uso de la TortugaBloques está disponible. (pdf)

Music Blocks

Music Blocks is a fork of Turtle Blocks that has programming elements specific to music.

A guide to Music Blocks is available. (pdf)

Una guia para los Bloques de Musica está disponible. (pdf)
