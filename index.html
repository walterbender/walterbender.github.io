Things you may enjoy:

Turtle Blocks

Turtle Blocks Javascript is an activity with a Logo-inspired graphical "turtle" that draws colorful art based on snap-together visual programming elements. Its "low floor" provides an easy entry point for beginners. It also has "high ceiling" programming, graphics, mathematics, and Computer Science features that will challenge the more adventurous student.

A guide to Turtle Blocks JS is available.

Music Blocks

Music Blocks is a fork of Turtle Blocks that has programming elements specific to music.
